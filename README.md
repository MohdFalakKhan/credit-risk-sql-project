## 📌 Project Overview
Analyzed 1,000 credit card transactions using MySQL to identify 
fraud patterns, risk trends, and anomalies using structured SQL queries.

**Dataset:** Credit Card Fraud Detection — Kaggle (MLG-ULB)

---

## 🛠️ Tools Used
- MySQL Workbench
- SQL (GROUP BY, WHERE, ORDER BY, CASE WHEN)
- Kaggle Dataset

---

## 📊 Key Findings
- Fraud rate = 0.2% (2 out of 1,000 transactions)
- Fraud avg amount (264.5) is 4x higher than normal (66.03)
- Fraud occurred in a specific time window (406–472)
- Amount alone is NOT sufficient to detect fraud
- Zero-value test transactions indicate classic fraud pattern
